## ✨ Features 
*  API Integration: Loaded all products from the API and rendered them on the UI.
*  Loading Spinner: I showed Loading Spinner for the late API Response.
*  Category-wise products: category-wise fetched products and rendered on the UI.
* Details Modal: quick modal shows above when clicking on the product card heading, and shows loading inside  before the product detail API response
* Cart management:  cart add, cart delete feature added, and saved them to the browser's local storage
* In-page Navigation: Made in-page navigation with smooth scroll for easy access to different sections
* Toastify: Used daisyUI toastify for add to cart and cart delete show




## 🛠️ Tech Stack

* HTML
* Tailwind CSS
* daisyUI 
* JavaScript (ES6+)
* API
